# Walk-Through Agent


https://github.com/user-attachments/assets/1ad0b126-cb58-48fe-8135-9ff95f3644b2

An intelligent walkthrough and guided tour agent for desktop applications built with **Tauri**. Create interactive, step-by-step user onboarding experiences for your desktop app with native accessibility support.

## Features

- 🎯 **Interactive Tours** – Guide users through your desktop app with customizable walkthroughs
- 🦀 **Tauri-based** – Cross-platform desktop application (Windows, macOS, Linux)
- ⚛️ **React UI** – Modern frontend for tour creation and management
- ♿ **Native A11y** – Swift integration for macOS accessibility API interaction
- 🎨 **Driver.js-inspired** – Similar highlight and tooltip mechanisms for desktop

## Tech Stack

- **Backend:** Rust (Tauri)
- **Frontend:** React
- **macOS A11y:** Swift
- **Cross-platform:** Tauri framework

## Use Cases

- User onboarding flows
- Feature discovery tours
- Tutorial walkthroughs
- In-app guidance and help

## Quick Start

```bash
yarn install
cargo tauri dev
```
